# Automated Panoramic Mapping of Virtual Worlds with a GeoGuessr-Style Platform for Apex Legends

A third year project for a Computer Science BSc at the
University of Warwick, supervised by [Dr Greg Watson](https://warwick.ac.uk/fac/sci/dcs/people/greg_watson/) and [Dr Alex Dixon](https://warwick.ac.uk/fac/sci/dcs/people/alex_dixon/)

## Abstract

> GeoGuessr is a popular online game in which players deduce geographic locations from panoramic imagery.
> It has inspired numerous adaptations, including those based on video games.
> However, these versions are limited, typically only offering a small set of guessable locations and lacking the use of panoramic imagery.
> These limitations arise due to the unfeasibility of manually collecting large-scale panoramic datasets.
> 
> To address this, the World Crawler was built, an automation tool capable of mapping virtual worlds into tens of thousands of panoramas.
> This level of scale was achieved by combining geometric image stitching with the properties of moddable video games, eliminating the need for more complex and traditional approaches.
> The tool was applied to Apex Legends, resulting in the generation of a dataset consisting of 25,633 locations across 13 worlds in only 37.4 hours, demonstrating its effectiveness.
> 
> Building on this, the project took the generated dataset and transformed it into a GeoGuessr-style platform for Apex Legends, called Apex Outlands.
> The development of this platform presents how these games are implemented, particularly the core gameplay aspect of playing matches.
> Concluding this component was a user study with 20 participants, highlighting the project’s potential impact on the Apex Legends community.
> 
> The results of this project show that, while the World Crawler is limited to moddable games, it provides the means to acquire large-scale panoramic data sets of these virtual worlds.
> This exemplifies its reach and how the work undertaken here can benefit communities beyond Apex Legends.
  
**Keywords:** *Computer Vision, Video Games, Apex Legends, GeoGuessr, Web Development, Python*
